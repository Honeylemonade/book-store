# book-store
> Spring Data JPA,Spring security with JWT, Spring Data Redis整合Demo
>

## 前端

1. Vue.js
2. router多级路由
3. Axios统一设置请求头添加token
4. localstorage
5. echart动态绘制图表
6. 悬浮购物车

## 后端

1. Spring security
   - 采用BCrypt加密
   - 权限细粒度为HttpMethod
   - 整合JWT
2. Redis
   - 采用Spring Data redis
   - key序列化器：StringRedisSerializer
   - value序列化器：Jackson2JsonRedisSerializer
3. Spring Data JPA
   - @Query注解自定义SQL
   - 采用JPQL
